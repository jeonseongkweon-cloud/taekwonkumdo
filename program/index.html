프로그램

구조 >> 교차원리, 교차원리 기반 무도를 교육 과정으로 구현합니다. “안전 → 구조 → 통제 → 기록 → 검증”이 하나로 연결됩니다.

교육 로드맵(국제 표준 흐름)

Core 1
기본 구조: 거리·각도·중심 고정 / 안전 규칙 습관화
Core 2
이동 구조: 이동/전환/회수 / 타이밍(시작-전개-회수)
Core 3
응용 구조: 상황 대응(통제) / 파트너 훈련의 표준
Core 4
심사 연계: 평가표 기반 실기+태도+안전 / 성장 기록

레벨 체계(예시 템플릿)

전 세계 지도자가 같은 용어로 지도/평가할 수 있도록 “레벨 언어”를 제공합니다.

Level A (Foundation)

기본 구조 / 안전 규칙 / 정확도

  • 기본 교차 구조
  • 정지·균형·시선
  • 도구 안전/보호

Level B (Movement)

이동·전환 / 타이밍 / 회수

  • 이동 교차
  • 회전·전환
  • 통제 기반 전개

Level C (Application)

응용 / 파트너 / 시나리오

  • 상황 판단
  • 거리/각도 선택
  • 안전 통제 종료

※ 레벨명/구성은 전총재님 공식 체계로 확정해 넣으면 완성도가 “국제급”으로 올라갑니다.

유소년 과정

안전 습관 + 집중력 + 기초 체력 + 구조 이해

안전 규칙기초 체력예절·태도성장 기록
  • 안전거리/보호구/공간 인식
  • 기본 자세·균형·리듬
  • 협동 훈련(파트너 규칙)
  • 월간 성장 체크(기록 기반)

성인 과정

구조·통제·건강·품격 — 지속 가능한 수련

정확도통제회수응용
  • 기본 교차(거리/각도/중심)
  • 이동 교차(전환/회수)
  • 시나리오 훈련(안전 통제)
  • 체력/유연성/부상 예방

지도자 과정(Leader Track)

표준 지도법 + 평가표 + 운영 규정 + 윤리

  • 지도자 표준 큐(코칭 언어)
  • 평가표 기반 피드백(국제 공통)
  • 사고 예방(리스크 관리)
  • 지부 운영 가이드(기본 프로토콜)

지도자 자료

세미나·공인행사

국가/지부 간 동일 표준으로 진행되는 공식 이벤트

  • 정기 세미나(기본/응용)
  • 지도자 워크숍(평가표/운영)
  • 공인 심사 연계 행사
  • 국제 교류 프로그램(파트너십)

행사 문의

심사 연계(공정성 설계)

심사는 “감”이 아니라 평가 프레임 + 표준 문서 + 기록으로 공정해집니다. 태권검도는 지도-심사-운영을 하나의 구조로 연결합니다.

1
자료실에서 기준 확인(규정/평가표)
2
현장 적용(지도자 과정/지부 표준)
3
평가/기록(영상·체크리스트)
4
검증(조회 서비스로 확인)

FAQ

프로그램을 도장에 도입하려면? 지정도장/지부 형태로 진행합니다.

운영 형태를 선택(지정도장/지부) 후, 자료실 기준을 확인하고 문의로 신청합니다.

지부 안내

지도자 과정은 어디서 확인하나요? 자료실/문의에서 최신 안내

과정 안내 문서와 제출 양식은 자료실에, 일정은 문의로 안내합니다.
