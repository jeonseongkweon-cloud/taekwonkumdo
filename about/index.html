연맹소개

구조 >> 교차원리, 교차원리 기반 무도 — 태권검도는 기술의 집합이 아니라 교육·심사·운영·안전·검증이 하나로 연결되는 “국제 표준 구조”입니다.

공식 선언(Official Statement)

“태권검도는 태권도의 장점과 검도의 장점을 구조적으로 통합하여, 안전·표준·기록·검증 가능한 국제 무도·스포츠 체계를 구축한다.”

정통성(창시자) 실행(운영 리더십) 표준(자료) 검증(조회) 확장(지부/언어)

명예총재 · 이사장(창시자) 인사말

Founder’s Message

사진

태권검도는 “멋”을 만들기 위한 기술이 아니라, 누구나 안전하게 배우고 바르게 성장할 수 있도록 움직임과 판단의 기준을 세우는 구조입니다.

저는 구조 >> 교차원리, 교차원리 기반 무도라는 프레임으로 거리·각도·타이밍·중심을 정리하고, 지도·심사·운영이 흔들리지 않게 표준을 만들고자 했습니다.

전통의 뿌리를 지키되, 시대의 도구(기록·분석·검증)를 활용해 태권검도가 세계 어디서든 동일한 품질로 전수되기를 바랍니다.

— 명예총재 · 이사장(창시자)

연맹 총재 인사말

President’s Message

사진

세계태권검도연맹은 철학을 현장에 구현하기 위해 교육 표준 · 지도자 인증 · 지부 운영 · 공정한 검증(조회)을 하나의 플랫폼으로 연결합니다.

태권검도의 강점은 “구조화”에 있습니다. 구조가 있으면 기술은 정돈되고, 지도는 표준화되며, 심사는 공정해지고, 국제 확장이 빨라집니다.

우리는 안전과 품격을 지키는 국제 네트워크를 만들겠습니다. 지부·지도자·행사·자료·검증이 하나로 이어지는 체계를 구축하겠습니다.

— 세계태권검도연맹 총재

비전 · 미션 · 핵심가치

Vision

태권검도를 “구조 기반 국제 표준 무도·스포츠”로 확립

Mission

교육 표준화 · 안전 원칙 · 지도자 인증 · 지부 확장 · 공정한 검증(조회)

Values

정통성 · 투명성 · 일관성 · 품격 · 국제 협력

거버넌스(운영 원칙)

표준 문서 기반 운영

규정·기준·양식·매뉴얼을 자료실에 공개하고, 업데이트 이력을 관리합니다.

공정한 심사/인증

심사 기준(평가표)과 인증 절차를 명확히 하여, 국가/지부 간 편차를 최소화합니다.

검증(조회) 우선

단증·자격·지정도장 정보를 조회로 확인할 수 있게 하여 신뢰를 설계합니다.
