연맹 소개

구조 >> 교차원리, 교차원리 기반 무도. 태권검도의 정통성과 운영 체계를 한 페이지에 정리합니다.

연맹 개요

  • 설립: 2009.08.08 창시 기반
  • 정의: 태권도의 장점 + 검도의 장점을 아우르는 무도·스포츠
  • 핵심 철학: 구조 >> 교차원리, 교차원리 기반 무도
  • 목표: 안전·기술·교육 표준화와 글로벌 네트워크 확장

명예총재 · 이사장(창시자) 인사말

Founder’s Message

사진

태권검도는 태권도와 검도의 장점을 구조적으로 통합한 신개념 무도입니다. 저는 구조 >> 교차원리, 교차원리 기반 무도를 기준으로 기술의 멋보다 안전, 속도보다 정확, 단기성과보다 ‘평생 성장’을 우선합니다.

전통의 뿌리를 잃지 않으면서도, 시대의 도구(AI 자세분석·기록·검증)를 활용해 누구나 올바르게 배우고, 바르게 성장하도록 길을 열겠습니다.

— 명예총재 · 이사장(창시자)

연맹 총재 인사말

President’s Message

사진

세계태권검도연맹은 교육 표준, 지도자 네트워크, 지부 운영, 공인 심사 체계를 통해 태권검도의 가치를 현장에 뿌리내리고 세계로 확장합니다.

우리는 ‘강함’을 보여주기보다 ‘안전하게 지켜내는 힘’을 만들겠습니다. 교육·행사·인증·기록이 하나로 연결되는 플랫폼을 구축하여 전 세계 어디서나 동일한 품질의 수련이 가능하도록 하겠습니다.

— 세계태권검도연맹 총재
